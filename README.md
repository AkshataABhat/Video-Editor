
## Set Up

Clone this repository
```bash
  git clone
```


**To run the code:**

```bash
  pip install moviepy
```
Include 3 video files as 1.mp4, 2.mp4, 3.mp4 in the same folder as videoeditor.py file
Also, include an audio file.

**Functionalities:**

1. Merge 
2. Crop
3. Resize
4. Speed
5. Colour
6. Trim
7. Audio


You can create a pull request to add any functionalities.
